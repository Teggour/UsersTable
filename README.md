Created with surge.sh

## **Getting started**

1. open project in terminal
2. cp .env.example .env
3. replace in .env file REACT_APP_API_TOKEN to your token from [gorest](https://gorest.co.in/) API
4. yarn or npm i
5. yarn run start or npm run start
